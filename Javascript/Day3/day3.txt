제어문

1. 조건문
-프로그램 내에서 주어진 조건문의 결과에 따라 별도의 명령을 수행하도록 제어하는 실행문
1) if(condition)
    if(조건식){ 참일경우 실행 }
    if(조건식){ 참일경우 실행 }else {거짓일경우 실행}

    if(조건식){
         참일경우 실행 
    }else if(조건식){
        위의 조건문이 거짓이고 다음 조건문이 참일경우 실행
    }else{
        모든 조건식이 거짓일 때 (생략가능)
    }

2) switch(변수){
    case 값1:
        변수의 값과 일치하는 경우실행할 문장
        break; 브레이크를 하지않으면 밑의 줄도 진행
    case 값2:

    default:
        비교값이 없다면 실행
}
