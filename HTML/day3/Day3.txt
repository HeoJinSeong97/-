비트맵과 벡터이미지

1. 비트맥 이미지
- 픽셀이 모여 만들어진 정보의 집합
- "레스터 이미지"라고 부름
- 픽셀 단위로 화면에 랜더링함.
- 그림판, 포토샵등 툴로 편집

2. 벡터 이미지
- 수학적 정보의 형태들이 만들어내는 결과물
- 이미지가 가지고 있는 점, 선, 면의 위치, 색상 정보를 가지고 있음
- 확대 및 축소를 해도 이미지가 깨지지 않음
- 일러스트 같은 툴로 편집

jpg(jpeg)
압출률이 훌륭하여 사진이나 예술분야에 많이 사용
- 가장 널리 쓰이는 이미지 포멧
- 표현 색상(24비트, 약 1600만 색상)이 뛰어나 고해상도 표시장치에 적합
- 손실 압축

gif
이미지 파일 내에 이미지 및 문자열 가은 정보등을 저장할 수 있는 파일
- 여러장의 이미지를 한 개의 파일에 저장할 수 있음(움짤, 애니메이션)
- 8비트(256색상) 컬러만 지원
- 비손실 압축

png
- gif의 대체 포멧으로 개발
- 8비트, 24비트 컬러 이미지 처리
- W3C 권장하는 포멧
- 비손실 압축
- 알파 채널 지원(투명도)

webp
- jpg, png, gif를 모두 대체할 수 있는 구굴이 개발한 이미지 포멧
- gif 같은 애니메이션 지원
- 알파 채널 지원(손실, 비손실 선택)
- 완벽한 포멧

*파일 경로 작성 방법
1. 절대경로
- 고유한 전체 경로
- http, https, file에 대한 경로(file경로는 사용하지 않음)

2. 상대경로
- html문서를 기준으로 경로
- 이미지가 HTML 문서와 같은 폴더에 있는 경우는 파일명만 씀
<img src="파일명">, <img src="./파일명">
- 이미지가 하위 폴더에 있는 경우 폴더명/파일명 씀
<img src="./폴더명/파일명" alt="강아지1">
- 이미지가 상위 폴더에 있는 경우
<img src="../파일명" alt="강아지1">
<img src="../폴더명/파일명" alt="강아지1">

테이블(Table)
테이블이란 여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표
<table>
    <tr> >>> 테이블의 각 층
        <th></th> >>>  테이블의 셀 중앙정렬
        <td></td> >>>  일반 왼쪽 정렬
    </tr>
</table>

<table> 속성
border : 테이블의 테두리 두깨를 설정(기본 0)
width : 테이블의 가로 폭을 설정
height : 테이블의 세로 폭을 설정
align : 테이블에 주면 테이블 위치가 정렬(left,right,center)

<tr>
테이블의 각 층을 구성
<th>
각 열의 제목
<th>
테이블의 셀

<tr><td><th> 속성
width : 셀의 가로 폭을 설정
height : 테이블의 세로 폭을 설정
align : 셀의 텍스트 정렬(left,right,center)

테이블 병합
colspan 속성
가로로 열을 합침
<td colspan="합칠 열의 갯수"

rowspan
세로로 행을 합침
<td rowspan="합칠 행의 갯수"

<colgroup>
<colgroup> 태그 뒤에 나오는 컬럽<th> 또는 <td>에 적용할 스타일을 해당 태그에서 미리 설정할 수 있도록 함

<cpation>
표에 제목을 붙일 떄 사용, 기본 위치는 표의 상단 중앙임


하이퍼링크 <a>
하이퍼 링크란 다른 페이지나 사이트로 연결하는 링크(문자 또는 이미지)
<a href = "사이트 또는 이동할 문서의 위치경로">링크에 사용될 문자 또는 이미지</a>
